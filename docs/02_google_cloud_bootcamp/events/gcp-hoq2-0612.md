## Description

生成 AI のワークロードが広がるなかで、GPU や TPU といったアクセラ レーターをいかに効率よく使い切るかが、推論基盤の設計における大きな論点になってきました。GKE を使えば、こうした貴重なリソースを無駄なく、必要な場面に賢く割り振れます。本セッションでは、Inference Gateway をはじめとする GKE 特有の機能を取り上げ、高性能な推論環境をどう組み立てるかを、実装ベースで学んでいきます。
