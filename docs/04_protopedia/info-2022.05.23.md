# 2022.05.23

## ライセンス情報追記と利用規約を変更しました

### **❏ライセンス情報追記について**

ProtoPediaは、登録いただいた作品がより多くの人の目に触れ、リアクションやフィードバックが生まれることで、登録いただいた皆様の次の創作意欲につながることを目指し運営をしております。 \
登録いただいた皆様の権利を保持しつつも、いろんな方に作品を紹介いただいたり、参考にしていただけるようにするため、登録作品情報にクリエイティブ・コモンズの「CCBYライセンス」を標準適用させていただきました。\
ライセンス表記は、コンテンツの柔軟な拡散と共有を促進するためのものです。ライセンス表記を追記すると、ユーザーがYou Tubeなどで作品を紹介することが可能になります。

2022年5月23日（月）に、下記のようにライセンス表記させていただきました。

**●ライセンス表記追記日 ：2022年5月23日（月）**

**●表示場所：作品公開日の横**

👇表示例

![](/files/ZXpktAWKf23UvxRf6AnE)

※CCライセンスは、著作権に関するライセンスで、作品を見た人に、ライセンスの条件に従った利用許可を与えるものです。作品情報（紹介画像、テキスト文章など）に対する適用であり、作品そのものの発明権利とは関係がありません。\
※CCBYライセンスは、原作者のクレジットを表示することを条件とし、コンテンツの改変を含むコピー・配布や、営利目的を含む使用も許可されるクリエイティブ・コモンズのライセンスです。\
　詳しくはこちら　→　<https://creativecommons.org/licenses/by/4.0/deed.ja>

<mark style="color:red;">**【重要：CCBYライセンスをしたくない方】**</mark>

作品情報をクリエイティブ・コモンズのCCBYライセンスにしたくない方は、ライセンス非表示の設定をお願いいたします。 \
ライセンス表示をされない場合、ユーザーは、法令で認められている範囲を超えたコンテンツの利用をすることはできません。

**●ライセンス非表示の設定方法** \
作品の編集画面の「ライセンス設定」で、「表示しない」を選択

![](/files/ipIuDdokAZMpb0gKHXms)

**●確認方法** \
以下のページで作品毎のライセンス表記を確認できます \
・設定ページ＞作品一覧（※ログインしてアクセスください）\
<https://protopedia.net/settings/prototypes>
